
# Python Todo List (CLI)

A simple command-line Todo List application written in Python.
This project was created to practice Python fundamentals and file handling.

## Features
- Add new tasks
- List all tasks
- Delete tasks
- Mark tasks as completed
- Persistent storage using a JSON file

## Technologies Used
- Python 3
- JSON

## How to Run
```bash
python todo.py
```
